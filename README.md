# Site Android

Página estática sobre a história do mascote do Android, desenvolvida com foco em HTML semântico, CSS responsivo e apresentação para portfólio.

## Sobre o projeto

O conteúdo apresenta a origem dos primeiros mascotes do Android, conhecidos como Dandroids, e a criação do Bugdroid, o robô verde associado ao sistema operacional.

## Tecnologias

- HTML5
- CSS3

## Pontos trabalhados

- Estrutura semântica com `header`, `nav`, `main`, `article`, `section`, `aside`, `figure` e `footer`
- Layout responsivo para celular, tablet e desktop
- Navegação por âncoras sem JavaScript
- Imagens com textos alternativos e carregamento otimizado
- Metadados básicos para SEO e compartilhamento
- CSS organizado com variáveis, media queries e estados de foco

## Como visualizar

Abra o arquivo `index.html` diretamente no navegador. O projeto não depende de servidor, JavaScript, frameworks ou ferramentas de build.
